/*	Folder Informations:						*/

The "help" folder is containing all MATLAB help files. These are
containing examples, tutorials, descriptions, ... which are included
to Matlab/Simulink.

-> IMPORTANT:	Before adding/modifying any *.html files or changing any
		*.xml files please read the basic Matlab help for
		creating/adding custom help files.

-> HELP URLS:	http://de.mathworks.com/help/matlab/matlab_prog/display-custom-documentation.html?refresh=true
		

-> Folders:	- html		This folder contains all help.html files
				and also the Table of Contens structure
				stored in helptoc.xml.
